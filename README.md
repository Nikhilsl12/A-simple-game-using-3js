# Fantasy Survival Arena

A stylized 3D fantasy survival game built using **Three.js** where the player controls a wizard fighting against flying enemies in an arena-style environment.

The project started as a simple frog-vs-bees prototype and gradually evolved into a fantasy-themed survival game with environmental generation, combat effects, collisions, and atmospheric world design.

---

## 🎮 Features

- 🧙 Wizard player character
- 🦇 Flying enemy AI
- ✨ Magical pulse attack system
- ❤️ 3-life survival system
- 🌲 Procedurally clustered environment
- 🏠 Village-style house placement
- 🌫️ Atmospheric fog and lighting
- 🪨 Tree and obstacle collision system
- 🎥 Isometric camera setup
- 🌍 Large explorable fantasy arena
- 🎨 Stylized low-poly visuals

---

## 🛠️ Tech Stack

- **Three.js**
- **JavaScript (ES6 Modules)**
- **Vite**
- **GLTF / GLB Models**
- **Sketchfab / Poly Pizza Assets**
